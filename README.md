## Ask My Agent Anything

Want to get to know me? Clone this repo and ask my agent questions about me,
answered from the garden's own pages and journals:

```bash
mise run amaa
```

That launches the agent, primed to answer. No [mise](https://mise.jdx.dev/)? Open
this folder in any AI coding tool (Claude Code, Cursor, Codex, Copilot) and just
ask — the AI configuration ships with the garden, including an `amaa` skill that
teaches the agent how to answer and cite its sources.

More detail lives on the
[Ask My Agent Anything page](https://codekiln.github.io/logseq-encode-garden/#/page/My%2FAI%2FAgent%2FAMAA).
