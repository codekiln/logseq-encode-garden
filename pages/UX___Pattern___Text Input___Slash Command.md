- # Slash Command
	- A text-input interaction pattern where typing `/` opens a filtered menu of actions, templates, or commands without leaving the current editing surface.
	- ## Why it is useful
		- Keeps action discovery close to the cursor.
		- Reduces memorization pressure for infrequent commands.
		- Blends freeform text entry with structured action selection.
	- ## Common interaction
		- A person types `/` inside a composer, editor, or other text input.
		- The interface shows a menu of available actions, often filtered as more characters are typed.
		- Choosing an item inserts or runs a command, template, workflow, or structured block.
	- ## Common homes
		- AI chat composers and coding-tool prompts.
		- Collaborative editors, notes apps, and issue trackers.
