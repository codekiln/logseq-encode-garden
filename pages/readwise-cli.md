- # [Readwise CLI: Everything You've Read, from the Command Line](https://readwise.io/cli)
	- Readwise's own official command-line tool for searching, reading, and organizing Readwise and [[Readwise/Reader]] data from the terminal.
	- ## Features
		- Hybrid (vector + full-text) search across the whole reading library
		- Retrieves saved articles, books, PDFs, podcasts, and newsletters as Markdown
		- Saves URLs and new documents to Reader
		- Manages tags, locations (inbox, archive, later, shortlist), and bulk edits
		- Creates, updates, and deletes highlights with notes
		- Daily spaced-repetition review of highlights
		- Exports entire libraries as Markdown files
		- Integrates with AI coding agents, including [[Claude/Code]], via pre-built workflow skills
	- ## Installation
		- `npm install -g @readwise/cli`
		- `readwise login` for browser-based OAuth, or `readwise login-with-token` for headless environments
