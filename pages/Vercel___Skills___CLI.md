- # Vercel Skills CLI
	- [vercel-labs/skills](https://github.com/vercel-labs/skills) · TypeScript · 21.4k ★
	- The open agent skills CLI for discovering and installing skills into 20+ AI coding tools. Run with `npx skills`.
	- Installs skills into [[Claude/Code]], Cursor, Codex, GitHub Copilot, Gemini CLI, Cline, and others.
	- Skills are security-audited before listing in the registry.
	- Backed by the browsable web registry at [[Vercel/Skills/Web]].
