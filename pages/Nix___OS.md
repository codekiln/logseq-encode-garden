- # NixOS
	- **Declarative Linux distribution** built on the [[Nix]] package manager: kernel, services, and user environments are defined in `.nix` configuration and are reproducible and rollback-friendly.
	- System modules and releases ship from the same upstream tree as [[Nix/Package/Registry]] (the `nixos/` subtree in `nixpkgs`).
	- [NixOS Manual](https://nixos.org/manual/nixos/stable/)
	- [Source repository](https://github.com/NixOS/nixpkgs) (includes `nixos/` system expressions)
