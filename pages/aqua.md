- # [aqua](https://aquaproj.github.io/)
	- Declarative **CLI version manager** (Go): pin dev tools in `aqua.yaml`, pull from GitHub releases and similar sources, enforce **checksums** and registry policy per the official docs.
	- Repository: [aquaproj/aqua](https://github.com/aquaproj/aqua).
	- Commonly reached through [[mise]] and other installers that use **aqua**-backed registries for verified OSS binaries.
