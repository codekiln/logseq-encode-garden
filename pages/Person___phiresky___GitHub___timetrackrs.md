- # [timetrackrs](https://github.com/phiresky/timetrackrs)
	- An automatic, rule-based time tracker: records activity as timestamped, tagged events and stores them in [[SQLite]] for analysis through a web dashboard.
	- Data sources include Linux X11 window tracking, browser history, VS Code, Android app usage, and shell history; a rule engine classifies events into tag hierarchies.
	- **Language:** [[Rust]] backend with a [[Typescript]] frontend.
