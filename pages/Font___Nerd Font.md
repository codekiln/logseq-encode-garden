- # Nerd Font
	- A font with extensive glyph support for terminal UI rendering and icon display.
	- Used by terminal applications like [[Ghostty]], [[yazi]], and [[Lazygit]] for rendering glyphs and UI elements.
	- Installed via [[brew/cask]] (e.g., `cask "font-jetbrains-mono-nerd-font"`) rather than package managers like [[mise]].
