# vim helphelp - the [[vim/help]] page on how to browse [[vim/help]] pages
	- to get here: [[vim/Keyshort/Get help on help]]
	- ## browsing vim help
		- [[vim/Keyshort/Help/Tag/Browse Back and Forward]]
			- use `CTRL-o` to go back, use `CTRL-t` to go forwards in [[vim/help/Tag]]s aka subjects.
		- [[vim/Keyshort/Help/Browse to or Search For Help Tag]]
			- use `:ta[g] <subject>` to browse to a named [[vim/help/Tag]]. `CTRL-o` goes back.
		- [[vim/Keyshort/Help/Follow a Help Tag Under Cursor]]
			- when "standing on" a help tag with the cursor, use `CTRL-]` to go there
		- [[vim/Keyshort/Help/Close a gO outline window]]
