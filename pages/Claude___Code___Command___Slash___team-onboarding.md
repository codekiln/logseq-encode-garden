- # Claude Code /team-onboarding
	- Generates a shareable onboarding guide from your Claude Code usage patterns.
	- The resulting link can be shared with teammates so they can onboard using your established workflows.
	- ## Usage
		- Run `/team-onboarding` inside a Claude Code session
		- Claude produces an `ONBOARDING.md` file summarizing your usage
		- The file is uploaded and a shareable URL is returned
