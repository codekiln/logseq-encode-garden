# OpenSpec Delivery Mechanisms
	- The delivery mechanisms determine how the OpenSpec framework will be "delivered" to the agentic harness, that is, how to register the OpenSpec commands / abilities in an [[AI/Coding/Tool]].
	- ## Mechanism 1 - [[Agent/Command]]s
		- This registers Slash Commands
			- [[Example]] in [[Claude Code Slash Commands]], `.claude/commands/opsx/{apply.md, archive.md, ...}`
	- ## Mechanism 2 - [[Agent/Skills]]
		- This will inject skills
