see-also:: [[tmux/Option/status-keys]], [[tmux/Mode]], [[tmux/Command/Prompt]]

- Entered with `Prefix :`.
- Replaces the status line (not the pane) with an input prompt.
- Key bindings controlled by [[tmux/Option/status-keys]] (`vi` or `emacs`).
- Also used by rename prompts (session, window) and search prompts.
