- # [hmans/miniplex](https://github.com/hmans/miniplex)
	- Developer-friendly entity management system for games and similarly demanding applications, built around [[Game/Video/Dev/ECS]] architecture.
	- Keeps entities as plain JavaScript objects with components as properties, which makes it lightweight and ergonomic in TypeScript-heavy codebases.
	- Leaves system scheduling to the host application instead of imposing its own runtime or engine loop.
