- # [TODOseq](https://github.com/scross01/obsidian-todoseq)
	- Lightweight, keyword-based task tracker for [[Obsidian]] with [[Logseq]] compatibility; scans the vault for state-keyword lines and displays them in a unified task panel.
	- **Stars:** 53 · **License:** MIT
	- Docs: [Introduction — Logseq compatibility](https://github.com/scross01/obsidian-todoseq/blob/main/docs/introduction.md)
	- ## Key features
		- Keyword-based task states: `TODO`, `DOING`, `DONE`, `LATER`, `NOW`, `WAIT`
		- Priorities and `SCHEDULED:` / `DEADLINE:` metadata (Logseq-compatible syntax)
		- Vault-wide unified task list panel
		- Designed for vaults shared with or migrated from [[Logseq]]
