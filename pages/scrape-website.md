- # [scrape-website](https://github.com/ventz/scrape-website)
	- Async website scraper that crawls an entire domain and downloads all pages (HTML), extracts clean [[Markdown]] (for LLMs/RAG knowledge bases), and saves documents (PDF, DOCX, XLSX, etc.). Stays within the target domain — never follows links to external sites.
	- Source repository: [ventz/scrape-website](https://github.com/ventz/scrape-website).
	- Written in [[Python]].
