- # `bat --style (-p)`
	- Controls which UI decorations bat renders around content
	- `-p` / `--style=plain` removes all chrome: no filename header, no line numbers, no grid borders
	- Other values: `full`, `header`, `numbers`, `grid` (combinable with commas)
