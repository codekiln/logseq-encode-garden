## [[CursorAI/Command]] and #Keyshort for showing the composer history #card
	- Command: **Show Composer History**
	- Keyshort: `⌥⌘L`
		- `⌥` (Option key)
		- `⌘` (Command key)
		- `L` (Letter L key)
	- [[VSCode/Keymap]]
		- ```json
		  {
		    "key": "alt+cmd+l",
		    "command": "composer.showComposerHistory"
		  }
		  ```
