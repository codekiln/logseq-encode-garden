- # [blueprint](https://numtide.github.io/blueprint/main/)
	- [[Nix]] project-structure library from [[Numtide]] that maps a standard folder layout to Nix flake outputs, so projects can split common outputs across predictable files and directories instead of concentrating glue in `flake.nix`.
	- The source repository is [numtide/blueprint](https://github.com/numtide/blueprint).
	- Supports common Nix ecosystem outputs and workflows including devshells, packages, [[Nix/OS]] configurations, nix-darwin, home-manager, system-manager, nix-unit, formatting, and flake checks.
	- Numtide summarizes it as a standard project structure for Nix.
