- # OpenVAS
	- ## Summary
		- OpenVAS (Open Vulnerability Assessment System) is an open-source software framework for vulnerability scanning and management.
		- It has been developed and maintained by [[Greenbone AG]] since 2006.
		- It is a core component of the [[Greenbone Vulnerability Management]] (GVM) suite.
		- It is widely used by security professionals to identify and mitigate security weaknesses in networks, applications, and systems.
	- ## Key facts
		- Developer: [[Greenbone AG]]
		- Initial release: [[2006]]
