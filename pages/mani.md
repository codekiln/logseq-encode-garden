- # [mani](https://manicli.com)
	- Go CLI for managing multiple [[git/repo]]s with declarative `mani.yaml`: clone/sync projects, run commands across tagged subsets, optional built-in TUI, filtering, completions; pitched for microservices and multi-repo setups.
	- Repository: [alajmo/mani](https://github.com/alajmo/mani). Official site [manicli.com](https://manicli.com).
	- Listed among [gitopolis alternatives](https://github.com/timabell/gitopolis#alternatives).
