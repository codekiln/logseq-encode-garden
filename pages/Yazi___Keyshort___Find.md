		- Shortcut: `/` to find, `ENTER` to confirm search query, `n` to find next, `N` to find previous
		- [[Example]]
			- `/md` search for md in any filename, which includes all markdown files in directory
			- `n` go to next occurrence
			- `N` to go to previous occurrence
			- ```
			   AGENTS.md  [1/3]                                         06/15 19:10
			    CLAUDE.md                                                06/15 19:10 
			    lefthook.yml                                             06/10 02:51 
			    mise.toml                                                06/21 10:05 
			   󰂺 README.md                                                06/14 11:02 
			    rulesync.jsonc                                           06/12 02:56 
			  ```
		- Description: Open find prompt searching forward (smart case).
		- [[My Notes]]
			- The official yazi docs list:
				- `/                find --smart              Find next file`
			- to me, this "Find next file" is a strange name for `/` which behaves more like "find file"
			- The "find next file" should be `n`, but I suppose that they are calling it find next file because it's position dependent
