- # Is there a keyboard shortcut in [[Obsidian]] for quickly finding keyboard shortcuts in Obsidian?
	- ## Answer
		- [[AI/Response]]
			- Obsidian does **not** document a dedicated “open shortcuts cheat sheet” hotkey whose only job is listing every shortcut. What it documents instead is **where to look from the keyboard**, quickly.
			- **Fastest per official help:** open the **Command palette** (default **Ctrl+P** on Windows/Linux, **Cmd+P** on macOS, unless you changed it). Obsidian’s Hotkeys help page states that the **fastest way to see the hotkey set for a command** is to find that command in the Command palette; if a hotkey exists, it is shown there.
			- **Full searchable list:** open **Settings → Hotkeys** (from the keyboard, **Ctrl+,** / **Cmd+,** opens Settings, then choose the **Hotkeys** section). That screen lists and filters **all** assignable commands and their bindings.
			- **Net:** the usual “in-app, from the keyboard” path is **palette first** (per-command shortcuts visible), **Settings → Hotkeys** when you want the complete filterable registry—not a separate bundled shortcut-reference panel.

				- [Obsidian Help — Hotkeys (source on GitHub)](https://github.com/obsidianmd/obsidian-help/blob/master/en/User%20interface/Hotkeys.md)
