- # [render-markdown.nvim](https://github.com/MeanderingProgrammer/render-markdown.nvim)
	- An [[nvim]] plugin that renders [[Markdown]] decorations inline using Neovim's virtual-text UI — headings, callouts, tables, and code blocks — with no external CLI and no separate terminal pane.
	- ## Key features
		- In-buffer rendering via Neovim's virtual-text API; no [[glow]] or other process required
		- Auto-updates as you type
		- Renders headings, callouts, tables, checkboxes, and code block backgrounds
