- # [Codex Goal Mode](https://developers.openai.com/codex/use-cases/follow-goals) - `/goal`
	- OpenAI's Codex introduced a feature called `/goal`.
	- Instead of prompting a model for a single response, you give the system an ongoing objective:
		- `/goal Complete [objective] without stopping until [verifiable end state].`
