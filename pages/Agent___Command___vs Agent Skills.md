# AI Agent Slash Comands vs AI Agent Skills
	- [[tldr]]
		- An [[AI/Agent]] can invoke [[Agent/Skills]] when it determines them to be relevant according to their description.
		- An [[AI/Agent]] will only invoke a [[Agent/Command]] if the human operator invokes it, or, in some cases, if the prompt instructions for a skill say to invoke that slash command.
	- In Claude Code, [[Agent/Skills]] can invoke [[Agent Commands]]. In other platforms, it may or may not be the same.
	- See Anthropic's docs comparing agent slash commands with [[Agent/Skills]] in [[Claude/Code/Command/Slash/Docs/Comparison with Skills]]
