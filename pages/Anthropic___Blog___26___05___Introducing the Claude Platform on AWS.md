- # [Introducing the Claude Platform on AWS](https://claude.com/blog/claude-platform-on-aws)
	- ## Summary
		- **GA** path for AWS customers to use the **full Claude Platform** with **AWS IAM**, **CloudTrail** audit logging, and **single AWS billing** (including commitment retirement); **Claude on Amazon Bedrock** remains available where **AWS** is the data processor.
		- Positions **Anthropic** as operator of the service with **native API parity** (features/betas ship same day as first-party API); contrasts with Bedrock for teams needing data processed **inside** AWS only.
	- ## Notes
		- Highlights include **managed agents**, **advisor strategy**, **web search/fetch**, **code execution**, **Files API**, **Skills**, **MCP connector**, **prompt caching**, **citations**, **batch**, plus **Claude Console** access.
