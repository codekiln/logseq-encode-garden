- # [wenlzhang/obsidian-file-title-updater](https://github.com/wenlzhang/obsidian-file-title-updater)
	- Community [[Obsidian]] plugin (**File Title Updater**) for the [[PTKM]] workflow; keeps a note's title aligned across filename, a configurable frontmatter title field, and the first level-1 heading.
	- Per-file commands pick the sync source (default, filename, frontmatter, or heading); folder context-menu actions bulk-sync markdown under a tree (irreversible—back up first).
	- Settings cover default source, which locations to sync, custom frontmatter field name, illegal-filename sanitization, and optional alias preservation when renaming (inspired by Smart Rename).
	- Docs: [ptkm.net/obsidian-file-title-updater](https://ptkm.net/obsidian-file-title-updater)
