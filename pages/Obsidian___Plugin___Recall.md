- # [Recall](https://github.com/martin-jw/obsidian-recall)
	- [[Obsidian]] community plugin for [[Flashcard/Spaced Repetition System|spaced repetition]] review using the [[FSRS]] ([[Flashcard/Spaced Repetition System/Free|Free Spaced Repetition Scheduler]]) algorithm.
	- Plugin ID: `obsidian-recall` · Author: martin-jw
	- ## Key features
		- FSRS algorithm — more modern and better-calibrated than SM-2
		- Supports note-level and card-level scheduling
		- Configurable target retention rate
