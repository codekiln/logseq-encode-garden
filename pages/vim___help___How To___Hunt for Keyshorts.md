# How to hunt for vim help on [[Keyshort]]s
	- The typical way that vim references help entries is using a syntax like
		- `CTRL-`
		- `META-`
		- For `g` then `CTRL-]`, try:
			- `:help g_CTRL-]`
	- These can be used with [[vim/help]], e.g. `:help CTRL-]` will take you to the help "jump to the definition of the keyword under the cursor. ..."
