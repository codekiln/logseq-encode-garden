- # SplxAI
	- [SPLX | End-to-End Security for AI](https://splx.ai/) - End-to-End Security for AI
		- Platform for automated security testing and continuous protection for enterprise AI systems
		- Features include:
			- Automated vulnerability scanning for conversational AI applications
			- Continuous security monitoring
			- Compliance with security standards
			- Probe - automates and continuously scans for vulnerabilities
		- Founded in 2023, headquartered in New York
		- In November 2025, SplxAI was acquired by Zscaler, a global cloud security leader
