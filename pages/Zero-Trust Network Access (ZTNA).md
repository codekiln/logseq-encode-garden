- # Zero-Trust Network Access (ZTNA)
	- Zero-Trust Network Access (ZTNA) is an access model that verifies user and device context before granting access to specific applications.
	- ZTNA is an alternative to [[VPN]] because it typically provides application-level access instead of broad network-level access.
	- ## Key idea
		- Apply least-privilege access per app or service.
		- Continuously evaluate identity, device posture, and policy.
		- Reduce exposed attack surface compared with full-tunnel remote access.
