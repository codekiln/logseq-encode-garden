- # App Switcher
	- In macOS, the `⌘ + Tab` pop-up is officially referred to as the **App Switcher**.
	- Sometimes described as the **application switcher interface**
	- It’s part of the macOS system UI for switching between running applications, distinct from:
		- **Mission Control** (all windows overview)
		- **Application Exposé** (all windows of the current app)
		- **Stage Manager** (window grouping feature in newer macOS versions)
