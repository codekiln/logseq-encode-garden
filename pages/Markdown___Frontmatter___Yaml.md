- # [[Markdown]] [[yaml]] Frontmatter
	- Many markdown processors support a type of frontmatter that can associate data with a page.
	- Typically it's inserted at the top of a markdown page like this:
		- ```
		  ---
		  key: value
		  key2: value2
		  ---
		  ```
