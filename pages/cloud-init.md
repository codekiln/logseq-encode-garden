- # [cloud-init](https://cloud-init.io/)
	- Industry-standard multi-distribution tool for cloud instance initialization. It reads cloud metadata plus optional user data and vendor data during boot, then configures Linux and Unix instances across public clouds, private cloud infrastructure, and bare-metal installs.
	- Applies user data automatically to customize instances: hostname, locale, SSH keys, mount points, packages, networking, and other boot-time setup.
	- [Source repository](https://github.com/canonical/cloud-init)
	- [Documentation](https://docs.cloud-init.io/en/latest/)
