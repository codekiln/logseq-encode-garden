- # [TOML](https://toml.io/)
	- Tom's Obvious, Minimal Language—a config file format for obvious semantics, UTF-8 text, and unambiguous mapping to hash tables.
	- Current standard: [TOML v1.0.0](https://toml.io/en/v1.0.0) (2021); specification and ecosystem work live under [toml-lang on GitHub](https://github.com/toml-lang).
	- Widely used for tool and application configuration (for example [[mise/Config/mise.toml]], Cargo, and Poetry).
