- # CIS Benchmarks
	- Community-developed secure configuration guidelines published by the [[Center for Internet Security]].
	- They provide prescriptive hardening recommendations across operating systems, cloud platforms, containers, network devices, web browsers, databases, mobile devices, and server software.
	- Benchmark profile levels:
		- `Level 1` is intended as a practical baseline with limited operational impact.
		- `Level 2` adds stricter controls for environments with higher security requirements.
	- Source:
		- [What are CIS Benchmarks? \| IBM](https://www.ibm.com/think/topics/cis-benchmarks)
