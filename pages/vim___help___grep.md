see-also:: [[vim/Keyshort/Get help on help]]

- # `:helpgrep` or `:helpg`
	- search for a term in the help
	- [[Examples]]
		- `:helpgrep CTRL-]`
			- view spots in the help that mention this keyboard shortcut
		- `:helpgrep jump back`
			- try to find the help on how to jump back to where you were, such as after a jump to a tag with `CTRL-]`
		- `:helpgrep tag stack`
		- `:helpgrep :tjump`
	- you'll get [[Vim/Err/E480]] if results do not exist.
