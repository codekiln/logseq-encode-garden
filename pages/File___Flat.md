- # Flat File formats
	- systems of organization that use plain text file formats like [[Markdown]] are usually set in opposition to systems that use [[Databases]]
	- ## advantages
		- [[git/Friendly]] - git is meant for text files
		- [[Intuit/ive]] - they work well with the [[Unix/Philosophy]]
		- usually they work well up to a certain project size, and then lead to performance problems. for example, [[LogseqDB]] is a response to this
