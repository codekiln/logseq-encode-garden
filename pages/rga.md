- # [ripgrep-all](https://github.com/phiresky/ripgrep-all)
	- `rga` wraps [[rg]] to search inside file types ripgrep cannot read directly — PDFs, e-books, Office documents, zip/tar.gz archives, SQLite databases, and more — by transparently extracting text before matching.
	- Invoked as `rga <pattern> <path>`, mirroring [[rg]] usage; results and flags carry over from ripgrep.
