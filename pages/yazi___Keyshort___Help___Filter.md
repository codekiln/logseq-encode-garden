- [[Keyshort]] [[yazi]] [[yazi/Keyshort]]
	- **Filter entries in the help menu** #card
		- Shortcut: `f`
		- Context: Help overlay (`[help]` layer, after opening help with `~` or `F1`)
		- Description: Narrows the help list by keyword (same idea as filtering the file list, but for help rows). Close or clear with `Esc` / `Ctrl+[` per preset. See [keymap.toml | Yazi — help.filter](https://yazi-rs.github.io/docs/configuration/keymap/) and [preset keymap-default.toml](https://github.com/sxyazi/yazi/blob/shipped/yazi-config/preset/keymap-default.toml).
