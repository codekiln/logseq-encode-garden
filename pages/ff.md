- # [ff](https://github.com/ff-notes/ff)
	- Distributed note taker and task manager using RON [[CRDT]]s.
	- it's pretty bare-bones:
		- ```
		  $ ff
		  Actual:
		    * buy milk
		      id 458a1456lrc-12z3opnykw, start 2018-01-10
		    * learn Haskell
		      id 459h2tqxutq-24nuhr86ae, start 2018-01-14
		  ```
