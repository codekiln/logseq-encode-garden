- # [Nunjucks](https://mozilla.github.io/nunjucks/)
	- A rich templating language for JavaScript, inspired by Jinja2. Supports template inheritance, macros, filters, async rendering, and a `{{ }}` / `{% %}` syntax familiar from Jinja2 and [[Mustache]].
	- Source repository: [mozilla/nunjucks](https://github.com/mozilla/nunjucks).
	- Used as one of the default template engines in [[Eleventy]].
