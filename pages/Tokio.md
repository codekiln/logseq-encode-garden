- # [Tokio](https://tokio.rs/)
	- An asynchronous [[Runtime]] for [[Rust]]. It provides what an async program needs to run: a multi-threaded task scheduler, async I/O, networking, timers, and synchronization tools.
	- [Source repository](https://github.com/tokio-rs/tokio) — [[GitHub/Star]] count: 32,391 as of [[2026-06-28 Sun]].
	- ## Notes
		- Built on Rust's `async`/`await`. Many Rust programs schedule their async work on it, including [[Deno]].
		- Distributed as the `tokio` [[Rust/crate]].
