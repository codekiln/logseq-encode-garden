- # [Cucumber](https://cucumber.io/)
	- A cross-language [[Behavior/Driven/Development]] tool for running automated acceptance tests written in plain language.
	- It executes feature specifications written in [[Gherkin]] and maps them to step definitions in the host language and test stack.
	- [[behave]] is one Python implementation in the wider Cucumber-style ecosystem.
