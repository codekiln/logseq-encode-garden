- # [nvim-ufo](https://github.com/kevinhwang91/nvim-ufo)
	- A [[nvim]] folding plugin that enhances the built-in fold experience with virtual text showing the number of folded lines, and support for LSP and tree-sitter as fold providers.
	- ## Key features
		- Multiple fold providers: LSP, tree-sitter, indent
		- Virtual text peek at folded line count
		- `zR` / `zM` remaps for open-all / close-all folds
		- Compatible with `foldmethod=expr`
