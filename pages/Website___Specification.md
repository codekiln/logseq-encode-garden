- # [The Website Specification](https://specification.website/)
	- Platform-agnostic, open-source specification of the technical features a good website should have. MIT licensed.
	- ## Normative links
		- [specification.website](https://specification.website/) — canonical spec site
		- [github.com/jdevalk/specification.website](https://github.com/jdevalk/specification.website) — source repository
	- ## Scope
		- Eleven areas: Foundations (HTML, encoding, OG, feeds), SEO (robots.txt, sitemaps, structured data), Accessibility (contrast, ARIA, keyboard nav), Security (HTTPS, CSP, security.txt), Well-Known URIs, Agent Readiness (/llms.txt, MCP, structured data), Performance (Core Web Vitals, caching, compression), Privacy (consent, GPC, data minimization), Resilience (error pages, service workers), Internationalization (hreflang, RTL), and Project resources.
	- ## Maintainer
		- Joost de Valk
