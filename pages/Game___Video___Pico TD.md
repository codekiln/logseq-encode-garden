- # [Pico TD](https://kicshikxo.itch.io/pico-td)
	- Pixel-art [[Game/Video/Tower Defense]] game built with [[Bevy]] in [[Rust]]; defend against incoming enemy vehicles using soldier units. Concept inspired by Bloons TD.
	- Sprites derived from [[Kenney]] asset packs (modified).
	- Three soldier classes (regular, rocket launcher, sniper) face land, sea, and air enemies across multiple levels; supports custom level upload via RON assets.
