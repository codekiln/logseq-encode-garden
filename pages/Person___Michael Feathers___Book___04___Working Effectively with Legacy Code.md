- # [Working Effectively with Legacy Code](https://www.amazon.com/Working-Effectively-Legacy-Michael-Feathers/dp/0131177052)
	- ## About
		- Book by [[Person/Michael Feathers]], published 2004
		- A practical guide to understanding, working with, and refactoring legacy codebases
		- Provides techniques and strategies for safely modifying code without comprehensive tests
		- Covers dependency breaking, characterization tests, and incremental refactoring approaches
	- ## Key Concepts
		- Legacy code is code without tests
		- Techniques for breaking dependencies to enable testing
		- Characterization tests to understand existing behavior
		- Seam identification and use
		- Incremental refactoring strategies
