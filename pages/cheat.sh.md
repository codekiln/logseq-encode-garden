- # [chubin/cheat.sh](https://github.com/chubin/cheat.sh)
	- Community-driven **cheat sheet** hub usable from the terminal (e.g. `curl cheat.sh/` / `curl cht.sh/`) and editors; aggregates many upstream doc sources (including [[tldr-pages]] pages) behind a single query interface.
	- **Lookup-oriented**: great for **public** command snippets, not for private editor keymaps unless you **mirror** or **generate** content yourself.
	- Companion namespace on the web is often **`cht.sh`** (same project).
