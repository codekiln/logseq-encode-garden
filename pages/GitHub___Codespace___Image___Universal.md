- # Universal image definition
	- The published **Microsoft Dev Container** **universal** image (notably `mcr.microsoft.com/devcontainers/universal`) is built from the Docker and feature setup under [devcontainers/images `src/universal` on `main`](https://github.com/devcontainers/images/tree/main/src/universal). That tree is the canonical source for what ships in the image (languages, CLIs, and preinstalled tooling) and is the same **kitchen-sink** base [[GitHub/Codespace]] commonly uses when a workspace is not pinned to a custom `image` in `.devcontainer`.
