- # [Jina AI Reader](https://r.jina.ai/)
	- Web-to-markdown conversion service from [[Jina AI]]. Prepend `https://r.jina.ai/` to any URL to retrieve a clean markdown version of that page, well-suited for feeding web content to LLMs.
	- ## Usage
		- `https://r.jina.ai/<url>` — returns markdown of the target page.
		- Example: `https://r.jina.ai/https://fzakaria.com/2026/06/01/every-byte-matters`
