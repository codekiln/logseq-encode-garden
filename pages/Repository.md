alias:: [[Repo]], [[Repositories]], [[Repos]]
- A repository is a storage location for a project's files along with the complete history of changes made to those files.
	- Repositories are the root abstraction of most [[Version/Control]] systems, such as [[git]], [[mercurial]], [[svn]], and [[cvs]].
	- A repository typically maps to a single project or codebase managed under [[source/control/management]].
	- ## Types
		- **Local repository** — a copy of the project on a developer's machine
		- **Remote repository** — a hosted copy, often on platforms like GitHub or GitLab
		- **Bare repository** — a repository without a working directory, typically used as a central hub
