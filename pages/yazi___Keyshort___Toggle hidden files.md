- [[Keyshort]] [[yazi]] [[yazi/Keyshort]]
	- **Toggle hidden files (dot-prefixed names)** #card
		- Shortcut: `.` (period)
		- Context: File list / manager pane (default keymap)
		- Description: Toggles whether entries starting with `.` are shown. Default-off visibility can be changed with `[mgr] show_hidden = true` in `yazi.toml`; see [yazi.toml | Yazi](https://yazi-rs.github.io/docs/configuration/yazi/) and [Quick Start](https://yazi-rs.github.io/docs/quick-start).
