- # [plugins-for-claude-natives](https://github.com/team-attention/plugins-for-claude-natives)
	- [[Claude Code]] plugin suite for power users by [[Team Attention]].
	- **Stars:** 802 · **License:** MIT · **Install:** `/plugin marketplace add team-attention/plugins-for-claude-natives`
	- ## Included plugins
		- **agent-council** — collect opinions from multiple AI models (Gemini, GPT, Codex) for consensus decisions
		- **clarify** — transform vague requirements into precise specifications via iterative questioning
		- **team-assemble** — dynamic agent team orchestration for complex tasks
		- **doubt** — force Claude to re-validate its own responses
		- **interactive-review** — web UI for visual plan/document approval
		- **dev** — developer workflow with community scanning and technical decision analysis
		- **session-wrap** — session analysis and validation toolkit
		- **youtube-digest** — summarize YouTube videos with transcripts and quizzes
		- **gmail** — multi-account Gmail integration
		- **google-calendar** — multi-account calendar with conflict detection
		- **say-summary** — text-to-speech summaries (Korean/English)
		- **podcast** — Korean podcast generation with YouTube auto-upload
		- **fetch-tweet** — fetch tweets without auth
		- **kakaotalk** — send/read KakaoTalk messages on macOS
