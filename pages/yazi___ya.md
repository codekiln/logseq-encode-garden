alias:: [[ya]]

- [`ya` CLI | Yazi](https://yazi-rs.github.io/docs/cli)
	- provides plugin management, [[yazi/Flavor]] management, DDS message publishing and subscribing
	- distributions include it by default when installing Yazi,
	- be sure that the versions of both `ya` and `yazi` are exactly the same.
