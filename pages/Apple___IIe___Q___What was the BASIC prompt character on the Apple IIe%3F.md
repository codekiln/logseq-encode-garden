- # What was the BASIC prompt character on the Apple IIe? [[card]]
	- ## [[AI Answer]]
		- [[AI/Response]]
			- **Short answer:** `]` — the closing bracket, used by **Applesoft BASIC**, which shipped in ROM on the Apple IIe.
			- The `>` character was the prompt for **Integer BASIC** (also called "Integer" or "Apple Integer BASIC"), the original BASIC on the Apple I and Apple II. Integer BASIC was not included in the Apple IIe ROM; it required a Language Card with the Integer ROM.
			- So both characters appear in Apple II history, but on the IIe the default power-on BASIC prompt was `]`.
			- See [Applesoft BASIC on Wikipedia](https://en.wikipedia.org/wiki/Applesoft_BASIC) and [Integer BASIC on Wikipedia](https://en.wikipedia.org/wiki/Integer_BASIC).
