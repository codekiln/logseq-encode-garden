- # [Front Matter Title](https://obsidian.md/plugins?id=obsidian-front-matter-title)
	- [[Obsidian]] community plugin by snezhig ([snezhig/obsidian-front-matter-title](https://github.com/snezhig/obsidian-front-matter-title)) that uses a frontmatter property value as the display title throughout the app.
	- ## Features
		- Reads a configurable frontmatter key (default: `title`) and substitutes its value as the displayed note title in tabs, the file explorer, graph view, and backlinks.
		- Allows the file's base name and the displayed title to differ — useful when filenames follow a machine-friendly convention but titles should be human-readable.
		- Title format is configurable: supports templates with fallback to the filename when the property is absent.
	- ## Installation
		- Search **Front Matter Title** in Settings → Community plugins → Browse.
