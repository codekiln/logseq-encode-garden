- # [Kanban Block](https://github.com/ldomaradzki/obsidian-kanban-block)
	- [[Obsidian]] community plugin that renders a `todo` fenced code block as an interactive three-column kanban board: `[ ]` → To Do, `[/]` → In Progress, `[x]` → Done (column names configurable).
