- # [skills.sh](https://www.skills.sh)
	- [github: vercel-labs/skills](https://github.com/vercel-labs/skills) · TypeScript · 21.4k ★
	- The open agent skills tool — a directory and CLI for discovering and installing agent skills across 20+ AI coding tools. Run with `npx skills`.
	- Skills are browsable by topic (React, Next.js, databases, testing, etc.), security-audited, and installable into Claude Code, Cursor, Codex, GitHub Copilot, Gemini, Cline, and others.
	- Competitor to [[skillshare]] (cross-agent skills sync) and [[rulesync]] (broader scope: rules, commands, subagents, MCP config in addition to skills).
	- See [[AI/Coding/Tool/Report/25/Skills Comparison]] for a comparative analysis; [[Claude Code/Skill/Project]] was the first prominent skills implementation in this ecosystem.
