- # [H5P](https://h5p.org/)
	- Open framework and packaging specification for interactive HTML5 content objects, distributed as self-contained `.h5p` ZIP files embeddable in any LMS or web page.
	- ## Scope
		- Defines a content-type framework, a JavaScript runtime API, and a package format (JSON manifest + assets in a `.h5p` ZIP) enabling portable, reusable interactive learning objects: quizzes, presentations, interactive video, branching scenarios, and 40+ other content types.
	- ## Maintainer
		- [[H5P Group]] (h5p.org); cloud-hosted platform at h5p.com.
	- ## Implementations
		- Reference open-source libraries on GitHub (h5p org).
		- Plugins available for [[Moodle]], [[WordPress]], [[Drupal]], and [[Canvas]].
	- ## Related standards
		- [[SCORM]], [[xAPI]], [[cmi5]], [[IMS/Common Cartridge]] — complementary e-learning interoperability standards. H5P objects are embeddable in SCORM or cmi5 packages; interactions can be tracked via xAPI.
