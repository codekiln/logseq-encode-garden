- # Suggest Completion
	- A text-input interaction pattern that proposes likely completions or matching items as a person types, so selection is faster than full manual entry.
	- ## Common interaction
		- A person begins typing into a text input.
		- The interface offers ranked completions, matches, or entities based on the current prefix and context.
		- Choosing a suggestion inserts the text, token, command, or linked object into the current surface.
	- ## Common homes
		- Skill pickers, mention menus, autocomplete fields, and command pickers.
