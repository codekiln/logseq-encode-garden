- # [Defuddle: Get the main content of any page as Markdown.](https://github.com/kepano/defuddle)
	- **JavaScript/TypeScript** extractor that takes a **URL or HTML**, finds the **main article body**, strips chrome like comments, sidebars, headers, and footers, and returns **cleaned HTML** or **Markdown**—positioned as a more forgiving alternative to Mozilla’s **Readability** with richer **metadata** (including schema.org) and more consistent handling of footnotes, math, and code blocks.
	- Ships as an **npm package** with **browser** (`defuddle`), **full** (`defuddle/full`), and **Node** (`defuddle/node` with any `Document` implementation such as linkedom or JSDOM) entry points, plus a **`npx defuddle parse …` CLI** (`--markdown`, `--json`, `--output`, etc.).
	- Originated for the **[Obsidian Web Clipper](https://github.com/obsidianmd/obsidian-clipper)** extension but is documented as **environment-agnostic**; README marks the project as **still a work in progress**.
