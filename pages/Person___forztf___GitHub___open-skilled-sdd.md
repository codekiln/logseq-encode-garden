- # [forztf/open-skilled-sdd](https://github.com/forztf/open-skilled-sdd)
	- Skill library for [[Claude/Code]] and other AI coding tools, created after studying [[Person/Brian Madison/GitHub/BMAD-METHOD]] and [[OpenSpec]].
	- Several skills ship with Chinese-language equivalents (e.g. `openspec-proposal-creation-cn`).
	- Uses [[EARS]] (Easy Approach to Requirements Syntax) for structuring requirements inside proposal/specification workflows.
	- ## Skills (via `skills.sh`)
		- `openspec-proposal-creation` — converts a feature request into an [[OpenSpec]]-style proposal with [[EARS]] requirements
		- `openspec-proposal-creation-cn` — Chinese-language variant of the above
		- `openspec-context-loading` — loads context needed for OpenSpec workflows
		- `openspec-implementation` — drives implementation from an approved proposal
		- `openspec-archiving` — archives completed OpenSpec work
