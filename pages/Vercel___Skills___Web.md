- # skills.sh — Vercel Skills Web
	- [skills.sh](https://www.skills.sh) — web interface for browsing the Vercel agent skills registry.
	- Skills browsable by topic: React, Next.js, databases, testing, and more.
	- The same registry that the [[Vercel/Skills/CLI]] (`npx skills`) draws from when discovering and installing skills.
