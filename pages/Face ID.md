- # Face ID
	- Face ID is Apple's facial biometric system for on-device user verification.
	- It is one form of [[Biometric]] authentication.
- ## Where It Fits
	- Commonly used as a local user-verification step for [[Passkey]], app unlock, and payment authorization.
	- In standards terms, it acts as user verification for a platform authenticator in [[FIDO2]] / [[WebAuthn]] flows.
