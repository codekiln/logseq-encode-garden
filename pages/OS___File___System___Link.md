- # File system link
	- A **file system link** is a name or pointer that lets one file-system location refer to file content or to another path.
	- The POSIX-core forms are [[OS/File/System/Link/Hard]] and [[OS/File/System/Link/Soft]].
	- The conceptual overview is [[OS/File/System/Link/Concept/Comparison of Different Types]].
