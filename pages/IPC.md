- # Inter-Process Communication (IPC)
	- Inter-Process Communication is the set of mechanisms that allow separate processes (programs or program instances) to exchange data and coordinate with each other, typically on the same machine or across a network. Common mechanisms include pipes, message queues, shared memory, semaphores, signals, and sockets.
