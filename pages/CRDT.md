- # Conflict-free replicated data type (CRDT)
	- ## Overview
		- [Conflict-free replicated data type - Wikipedia](https://en.wikipedia.org/wiki/Conflict-free_replicated_data_type)
			- > The CRDT concept was formally defined in [[2011]] by Marc Shapiro, Nuno Preguiça, Carlos Baquero and Marek Zawirski
			- ### [Industry use](https://en.wikipedia.org/wiki/Conflict-free_replicated_data_type#Industry_use)
				- [[Microsoft]] uses it in [[Fluid/Framework]], an SDK for making real-time collaboration apps like [[o365]]
				- [Phoenix](https://en.wikipedia.org/wiki/Phoenix_(web_framework)), a web framework written in [Elixir](https://en.wikipedia.org/wiki/Elixir_(programming_language)), uses CRDTs to support real-time multi-node information sharing in version 1.2.
				- [Apple](https://en.wikipedia.org/wiki/Apple_Inc.) implements CRDTs in the Notes app for syncing offline edits between multiple devices.
