- # [FastMCP](https://gofastmcp.com/)
	- Repository: [PrefectHQ/fastmcp](https://github.com/PrefectHQ/fastmcp)
	- [[GitHub/Star]]: 26,176 (checked 2026-07-13).
	- The fast, Pythonic way to build [[MCP]] servers, clients, and apps — expose Python functions as MCP-compliant tools, resources, and prompts, with schema generation, validation, transport negotiation, and protocol lifecycle handled automatically.
	- Written in [[Python]].
	- Maintained by [[Prefect]] and underpins its [[Prefect/Horizon]] platform.
	- ## Versions and the MCP Python SDK
		- **FastMCP 1.0** was incorporated into the official MCP Python SDK in 2024.
		- **FastMCP 2.0** is the actively maintained standalone project, which Prefect cites as powering a large share of MCP servers across languages.
