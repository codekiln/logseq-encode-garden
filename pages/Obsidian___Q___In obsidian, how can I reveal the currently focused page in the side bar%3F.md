	- ## [[My Answer]]
		- cmd + P then search for "Files: Reveal current file in navigation" by searching for "Reveal"
	- ## [[AI Answer]]
		- **Short answer:** Use the command **"Files: Reveal current file in navigation"** — no default hotkey, but you can assign one in Settings → Hotkeys.
		- **Via Command Palette:** Press `Cmd+P` (macOS) / `Ctrl+P` (Windows/Linux), type `reveal`, and select **Files: Reveal current file in navigation**. The File Explorer sidebar will scroll to and highlight the active file.
		- **Via right-click:** Right-click the active tab title and choose **Reveal file in navigation**.
		- **Assign a hotkey:** Settings → Hotkeys → search "Reveal current file in navigation" → click the `+` to bind a shortcut (e.g. `Ctrl+.`).
		- **Auto-reveal (community plugin):** The [Auto Reveal in Explorer](https://community.obsidian.md/plugins/auto-reveal-in-explorer) plugin automatically scrolls the File Explorer to the active file whenever you switch tabs — no manual command needed.
