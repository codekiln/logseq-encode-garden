	- **Set or clear star rating with number keys (Library)** #card
		- Set rating: `1` through `5` (applies that many stars)
		- Set rating and go to next photo: `Shift + 1` through `Shift + 5`
		- Remove star rating (clear to zero stars): `0`
		- Remove star rating and go to next photo: `Shift + 0`
		- Context: Library module; defaults per [Adobe's shortcut list](https://helpx.adobe.com/lightroom-classic/help/keyboard-shortcuts.html).
		- Description: Direct numeric star assignment and clearing during culling.
