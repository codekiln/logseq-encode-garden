- [[Keyshort]] [[tmux]] [[tmux/Keyshort]] [[tmux/Keyshort/Window]]
	- **Select next layout** #card
		- Shortcut: `<tmux-prefix> Space` (default prefix: `Ctrl+b`, so `Ctrl+b` then `Space`)
		- Description: Cycles the current window through tmux **layout** presets (for example even-horizontal vs even-vertical), redrawing how the split dividers run while keeping the same panes
		- Note: This changes **layout geometry**, not pane focus — contrast [[tmux/Keyshort/Pane/Rotate Panes]] (`<prefix> Ctrl+o`), which rotates which pane sits where
