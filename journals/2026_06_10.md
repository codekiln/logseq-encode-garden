- [[TIL]]
	- [[MCP]] (Model Context Protocol) is a pun on MCP — the Master Control Program — from the 1982 Tron movie, where MCP was the villainous AI that sought to dominate all computer programs
	- Skills can be bundled into an [[MCP/Server]], making them available to any MCP client (not just Claude Code)
	- [[MCP/Server]] deployments that scale to zero need a token cache layer — e.g. Redis, KV, DynamoDB, or [[Cloudflare/DurableObjects]] — since in-process token state is lost on each cold start
	- Async server-side agents calling protected APIs need [[Auth/M2M]] (client credentials / short-lived JWT) rather than user-delegated flows, since there is no human present to authorize
