		- [[Claude Code]] permission prompts: when Claude asks for permission, option 2 is often a very specific command — you can select it and then use the delete key to broaden it. For example, `bd --no-daemon --specific-flags ...` becomes `bd --no-daemon *`, and the entry saved to `.claude/settings.local.json` is `"Bash(bd --no-daemon *)"`. See [[Claude Code/How To/Answer Permission Dialog To Record Wildcard Permissions in Local Settings]].
		- ==text== (double-equals) highlighting in Logseq is not standard Markdown — it's a Pandoc extension that renders as HTML `<mark>`. Obsidian also supports it, but CommonMark does not.
		- in [[tmux]] (or [[oh-my-tmux]] — not sure which), you can drag the divider between two panes with the mouse to resize them live. Mouse mode must be enabled; the feature likely comes from `set -g mouse on` in `.tmux.conf`, which hands pointer events to the terminal multiplexer so it can interpret drag gestures on pane borders.
		- [github.com/codespaces](https://github.com/codespaces) lists all your [[GitHub]] Codespaces and lets you request a usage report for each one.
		- [github.com/codespaces](https://github.com/codespaces) also lets you export unpushed changes to a new branch before deleting a Codespace — so you don't lose work when cleaning up.
