				- status-line `#()` hooks (for example, right part of status bar could be a notification center, but this doesn't really scale well)
				- [[tmux/rename-window]] - problem is that if an agent does this in the background, it renames whatever window is focused in the session, not the window that the agent is in
					- `BEAD_ID` / direnv) - in theory, [[direnv]] could be scripted to set an environment variable like `BEAD_ID` and then that could be used to help the agent call [[tmux/rename-window]]
				- dashboard
					- use unix tool `watch` + `bd list`, and [[gum]] or [[fzf]] for menus
					- how to achieve [[Kanban]]-ish views of [[bd]] tasks?
